3 — 10 Melhorias propostas

Marque as que quer aplicar — ou diga "todas" para o pacote completo.

1
Easing curves premium (cubic-bezier) nos tokens
Adiciona --ease-out-expo, --ease-spring, --ease-in-out-quart no tokens.css + substitui ease genérico nas variáveis de transição. Base para todas as outras melhorias.
visual ALTO esforço BAIXO
2
Animação de entrada de página (fadeSlideIn)
Keyframe opacity 0→1 + translateY 10px→0 de 320ms aplicado no <main> do AppShell sempre que a rota muda. Elimina o corte brusco ao navegar entre módulos.
visual ALTO esforço BAIXO
3
Submenu sidebar com stagger (slide+fade por item)
NavItemGroup abre/fecha com max-height transition + opacity + translateY por item com delay de 40ms entre eles. Actualmente aparece/desaparece instantaneamente.
visual ALTO esforço MÉDIO
4
KpiCard hover lift + sombra elevada
Adiciona translateY(-3px) + shadow-md no hover do KpiCard. Atualmente o card não reage ao mouse de forma alguma.
visual ALTO esforço BAIXO
5
Botão primary com gradiente + spring bounce no hover
Substitui a cor sólida por linear-gradient(135deg, #3b82f6, #2563eb) e adiciona translateY(-2px) + box-shadow glow no hover com ease-spring.
visual ALTO esforço BAIXO
6
Focus-visible ring em todos inputs, selects, buttons
Adiciona :focus-visible { box-shadow: 0 0 0 3px rgba(59,130,246,.25) } no globals.css para inputs e outline: 2px solid var(--accent) em buttons. Acessibilidade + polimento.
visual MÉDIO esforço BAIXO
7
Radius upgrade: rounded-sm → rounded-md em cards
KpiCard, cards de módulo e modais sobem de border-radius: 4px para 6-8px. Sidebar e botões ficam em 4px (mais sharp = intencional). Diferencia hierarquia visual de container vs controle.
visual MÉDIO esforço BAIXO
8
Stagger de entrada no KpiGrid
Cada card do KpiGrid entra com delay de 60ms em cascata via animation-delay: calc(var(--i) * 60ms). Dá a sensação de que os dados "chegam" na tela, não "já estavam lá".
visual ALTO esforço MÉDIO
9
Sidebar active item: inner glow sutil
Item ativo ganha box-shadow: inset 0 1px 0 rgba(59,130,246,.15), inset 0 -1px 0 rgba(59,130,246,.08). Reforça visualmente que aquele item está "afundado/selecionado" sem adicionar ruído.
visual MÉDIO esforço BAIXO
10
Scrollbar custom + prefers-reduced-motion global
Scrollbar fina (6px) com thumb #334155 e hover #475569. Bloco @media (prefers-reduced-motion: reduce) desativando todas as animações. Toque de acabamento e acessibilidade obrigatória.
visual MÉDIO esforço BAIXO
